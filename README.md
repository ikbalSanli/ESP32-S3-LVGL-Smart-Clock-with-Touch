ESP32-S3 LVGL Smart Clock

This project is a touch-enabled smart clock built on ESP32-S3 Touch LCD (1.28”) using the LVGL graphics library.

Features

Real-time clock with WiFi & NTP synchronization

Touch screen support (CST816S)

Animated seconds arc and modern LVGL UI

WiFi status indicator

Runs smoothly with partial screen refresh

Technologies

ESP32-S3

LVGL

TFT_eSPI

WiFi & NTP

I2C Touch Controller (CST816S)

Hardware

ESP32-S3 Touch LCD 1.28"

240×240 round display

Demo

The clock updates every second and synchronizes time automatically when WiFi is available.
